Pengertian OOP

Object Oriented Programming (OOP) adalah metode pemrograman yang menggunakan konsep objek. Dalam OOP program dibangun dari kumpulan objek yang memiliki data dan fungsi. Konsep ini membantu membuat program lebih terstruktur dan mudah dikembangkan.

Encapsulation

Encapsulation adalah proses membungkus data dan method dalam satu class agar data tersebut terlindungi dan tidak dapat diakses secara langsung dari luar class.

class Mahasiswa { String nama; int umur; void tampilkan(){ print(nama); } }

Inheritance

Inheritance adalah konsep pewarisan dimana sebuah class dapat mewarisi atribut dan method dari class lain sehingga kode dapat digunakan kembali.

class Hewan{ void makan(){ print("Hewan sedang makan"); } } class Kucing extends Hewan{ }

Polymorphism

Polymorphism adalah kemampuan objek untuk memiliki banyak bentuk sehingga satu method dapat digunakan dengan cara yang berbeda.

class Hewan{ void suara(){ print("Suara hewan"); } } class Kucing extends Hewan{ void suara(){ print("Meong"); } }
