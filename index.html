Penjelasan Konstruktor dan Enkapsulasi

Dalam pemrograman berorientasi objek (OOP), terdapat konsep penting yang membantu mengatur struktur program agar lebih rapi dan mudah dipahami. Dua konsep tersebut adalah konstruktor dan enkapsulasi.

Konstruktor merupakan method khusus yang digunakan untuk membuat atau menginisialisasi objek dari sebuah class. Sedangkan enkapsulasi adalah teknik untuk melindungi data dalam class agar tidak dapat diakses secara langsung dari luar.

Contoh Named Constructor

Named constructor memungkinkan sebuah class memiliki lebih dari satu constructor dengan nama berbeda sehingga objek dapat dibuat dengan cara yang berbeda sesuai kebutuhan.

DART SOURCE CODE
// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

class Mobil {
  String? merk;
  int? tahun;

  Mobil(this.merk, this.tahun);

  Mobil.baru() {
    merk = 'Toyota';
    tahun = 2023;
  }

  Mobil.dariJson(Map json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }

  String toString() {
    return 'Mobil(merk: $merk, tahun: $tahun)';
  }
}

void main() {
  var mobilA = Mobil('Honda', 2020);
  print(mobilA);

  var mobilB = Mobil.baru();
  print(mobilB);

  var mobilC = Mobil.dariJson({"merk":"Suzuki","tahun":2021});
  print(mobilC);
}

Kesimpulan

Constructor digunakan untuk menginisialisasi objek ketika sebuah class dibuat. Dalam bahasa Dart terdapat beberapa jenis constructor seperti default constructor, named constructor, dan constant constructor.

Dengan menggunakan konsep enkapsulasi, data dalam class dapat dilindungi sehingga program menjadi lebih aman dan terstruktur.
