Penjelasan Konstruktor dan Enkapsulasi

Dalam pemrograman berorientasi objek (OOP), terdapat beberapa konsep penting untuk mengatur struktur program agar lebih rapi dan mudah dipahami. Dua konsep yang sering digunakan adalah konstruktor dan enkapsulasi.

Konstruktor merupakan method khusus dalam class yang digunakan untuk membuat atau menginisialisasi objek. Sedangkan enkapsulasi adalah teknik untuk melindungi data dalam class agar tidak dapat diakses langsung dari luar.

Named Constructor (Class Mobil)

Named constructor memungkinkan sebuah class memiliki lebih dari satu constructor dengan nama berbeda.

DART SOURCE CODE
// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

import 'dart:convert';

class Mobil {
  String? merk;
  int? tahun;

  Mobil(this.merk, this.tahun);

  Mobil.baru() {
    merk = 'Toyota';
    tahun = 2023;
  }

  Mobil.dariJson(Map json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }

  String toString() {
    return 'Mobil(merk: $merk, tahun: $tahun)';
  }
}

void main() {
  var mobilA = Mobil('Honda', 2020);
  print(mobilA);

  var mobilB = Mobil.baru();
  print(mobilB);

  var mobilC = Mobil.dariJson({"merk":"Suzuki","tahun":2021});
  print(mobilC);
}

Constant Constructor (Class Warna)

Constant constructor digunakan untuk membuat objek yang bersifat konstan atau tidak berubah.

DART SOURCE CODE
// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

class Warna {
  final int r, g, b;

  const Warna(this.r, this.g, this.b);

  String toString() {
    return 'Warna(r: $r, g: $g, b: $b)';
  }
}

void main() {
  const merah = Warna(255,0,0);
  const merahLagi = Warna(255,0,0);

  print(merah);
  print(merahLagi);
}

Initializer List (Class Titik)

Initializer list digunakan untuk menginisialisasi nilai variabel sebelum constructor dijalankan.

DART SOURCE CODE
// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A

class Titik {
  final double x, y;

  Titik(double x, double y)
      : x = x.abs(),
        y = y.abs();
}

void main() {
  var titik = Titik(-5,10);
  print("x: ${titik.x}, y: ${titik.y}");
}

Kesimpulan

Constructor digunakan untuk menginisialisasi objek saat sebuah class dibuat. Dalam bahasa Dart terdapat berbagai jenis constructor seperti default constructor, named constructor, dan constant constructor.

Konsep enkapsulasi digunakan untuk melindungi data dalam class dengan menggunakan private variable, getter, dan setter sehingga program menjadi lebih aman, terstruktur, dan mudah dikelola.
