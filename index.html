LAPORAN OOP

Pengertian OOP

Object Oriented Programming (OOP) adalah metode pemrograman yang berfokus pada objek. Program dibangun menggunakan class dan object yang memiliki data (atribut) dan perilaku (method). Tujuannya agar program lebih terstruktur, mudah dikembangkan, dan mudah dipahami.

Class Hewan

Class adalah cetakan untuk membuat object. Object adalah hasil dari class yang dapat digunakan dalam program. 

class Hewan {
  String nama;

  Hewan(this.nama);

  void bersuara() {
    print("$nama sedang bersuara...");
  }
}

Class Mahasiswa

Latihan ini bertujuan memahami pembuatan class dengan beberapa atribut. 

class Mahasiswa {
  String nama;
  String nim;
  double ipk;
}

Constructor

Constructor adalah method khusus yang digunakan untuk memberi nilai awal pada object saat dibuat. 

class Car {
  String? model;
  String? color;

  Car(this.model, this.color);
}

Encapsulation

Encapsulation adalah konsep untuk melindungi data dalam class agar tidak bisa diakses langsung dari luar. 

class Person {
  String _name;
  int _age;
}

This Keyword

Kata kunci this digunakan untuk merujuk pada object yang sedang aktif di dalam class. 

class Product {
  String nama;

  Product(this.nama);
}

Class Produk

Latihan membuat class dengan constructor dan beberapa atribut. 

class Produk {
  final String id;
  final String nama;
  final double harga;
}
