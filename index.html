Penjelasan Konstruktor dan Enkapsulasi

Dalam pemrograman berorientasi objek (OOP), terdapat beberapa konsep penting untuk mengatur struktur program agar lebih rapi dan mudah dipahami. Dua konsep yang sering digunakan adalah konstruktor dan enkapsulasi. Konstruktor merupakan method khusus dalam class yang digunakan untuk membuat atau menginisialisasi objek. Sedangkan enkapsulasi adalah teknik untuk melindungi data dalam class agar tidak dapat diakses langsung dari luar, biasanya menggunakan private variable, getter, dan setter.

Jenis-jenis Konstruktor Dart

Default Constructor, Named Constructor, Parameterized Constructor, Constant Constructor.

Named Constructor (Class Mobil)

Constructor digunakan untuk menginisialisasi objek ketika class dipanggil. Pada Dart juga terdapat named constructor yang memungkinkan sebuah class memiliki lebih dari satu constructor dengan nama berbeda. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  
import 'dart:convert';

class Mobil {
  String? merk;
  int? tahun;

  Mobil(this.merk, this.tahun);

  Mobil.baru() {
    merk = 'Toyota';
    tahun = 2023;
  }

  Mobil.dariJson(Map json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }

  String toString() {
    return 'Mobil(merk: $merk, tahun: $tahun)';
  }
}

void main() {
  print('--- Demonstrasi Class Mobil ---');

  var mobilA = Mobil('Honda', 2020);
  print('Mobil A: $mobilA');

  var mobilB = Mobil.baru();
  print('Mobil B: $mobilB');

  String jsonString = '{"merk": "Suzuki", "tahun": 2021}';
  Map jsonData = jsonDecode(jsonString);

  var mobilC = Mobil.dariJson(jsonData);
  print('Mobil C: $mobilC');

  Map anotherJsonData = {
    'merk': 'Mitsubishi',
    'tahun': 2022
  };

  var mobilD = Mobil.dariJson(anotherJsonData);
  print('Mobil D: $mobilD');

  print('--- Akhir Demonstrasi ---');
}

Constant Constructor ( Class Warna )

Constant constructor digunakan untuk membuat objek yang bersifat konstan atau tidak berubah. Constructor ini biasanya digunakan pada class yang semua atributnya bertipe final. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  
class Warna {
  final int r, g, b;

  const Warna(this.r, this.g, this.b);

  String toString() {
    return 'Warna(r: $r, g: $g, b: $b)';
  }
}

void main() {
  const merah = Warna(255, 0, 0);
  const merahLagi = Warna(255, 0, 0);

  print(merah);
  print(merahLagi);
}

Initializer List ( Class Titik )

Initializer list digunakan untuk menginisialisasi nilai variabel sebelum constructor dijalankan. Biasanya digunakan pada variabel yang bertipe final. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  
class Titik {
  final double x, y;

  Titik(double x, double y)
      : x = x.abs(),
        y = y.abs() {
    print('Titik dibuat di (${this.x}, ${this.y})');
  }
}

void main() {
  print('Membuat objek Titik pertama:');
  var titik1 = Titik(3.0, 4.0);
  print('Titik 1: x=${titik1.x}, y=${titik1.y}');

  print('\nMembuat objek kedua dengan nilai negatif:');
  var titik2 = Titik(-5.0, 10.0);
  print('Titik 2: x=${titik2.x}, y=${titik2.y}');

  print('\nMembuat objek ketiga dengan nilai negatif di kedua koordinat:');
  var titik3 = Titik(-1.5, -2.5);
  print('Titik 3: x=${titik3.x}, y=${titik3.y}');
}

Class Produk ( Getter & Setter Validasi )

Pada contoh ini, getter dan setter digunakan untuk mengatur akses terhadap data produk serta menambahkan validasi agar nilai yang dimasukkan tetap sesuai aturan. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  
class Produk {
  String _nama;
  double _harga;

  Produk(this._nama, this._harga);

  String get nama => _nama;
  double get harga => _harga;

  set nama(String value) {
    if (value.length >= 3) {
      _nama = value;
    } else {
      print('Gagal memperbarui nama: minimal 3 karakter.');
    }
  }

  set harga(double value) {
    if (value > 0) {
      _harga = value;
    } else {
      print('Gagal memperbarui harga: harus lebih dari 0.');
    }
  }
}

void main() {
  var produk1 = Produk('Monitor Samsung', 3500.0);

  print('Nama Produk: ${produk1.nama}');
  print('Harga Produk: Rp ${produk1.harga}');

  produk1.nama = 'Keyboard Mekanik';
  produk1.harga = 1200.0;

  print('Nama Produk baru: ${produk1.nama}');
  print('Harga Produk baru: Rp ${produk1.harga}');
}

Class User ( Login & Ganti Password )

Contoh ini menunjukkan penerapan enkapsulasi pada sistem login sederhana, dimana username dan password disimpan sebagai variabel private dan proses login dilakukan melalui method dalam class. 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
class User {
  String _username;
  String _password;

  User(this._username, this._password);

  String get username => _username;

  bool login(String username, String password) {
    return _username == username && _password == password;
  }

  void gantiPassword(String lama, String baru) {
    if (_password == lama) {
      if (baru.length >= 8) {
        _password = baru;
        print('Password berhasil diubah.');
      } else {
        print('Password baru minimal 8 karakter.');
      }
    } else {
      print('Password lama tidak sesuai.');
    }
  }
}

void main() {
  final user1 = User('admin_user', 'securePass123');

  print('User: ${user1.username}');

  if (user1.login('admin_user', 'securePass123')) {
    print('Login berhasil');
  } else {
    print('Login gagal');
  }

  user1.gantiPassword('securePass123', 'newSecurePass456');
}

Kesimpulan

Berdasarkan materi yang dipelajari, dapat disimpulkan bahwa constructor digunakan untuk menginisialisasi objek saat sebuah class dibuat. Dalam bahasa Dart terdapat beberapa jenis constructor seperti default constructor, named constructor, dan constant constructor yang dapat digunakan sesuai kebutuhan program. Selain itu, konsep enkapsulasi digunakan untuk melindungi data dalam sebuah class dengan membatasi akses langsung terhadap variabel menggunakan private variable, getter, dan setter. Dengan menerapkan konsep ini, program menjadi lebih terstruktur, aman, dan mudah dikelola.

Tugas 1 Bonus 2: Class Rekening Bank

Program ini membuat kelas RekeningBank menggunakan konsep enkapsulasi dalam OOP. Kelas ini memiliki field private _nomorRekening dan _saldo. Terdapat method deposit(), tarik(), dan cekSaldo() untuk mengelola saldo dengan aman sehingga saldo tidak bisa menjadi neg 

// Nama  : Putri Ananda Amelia
// NIM   : 251410029
// Kelas : SI2A
  
class RekeningBank {
  String _nomorRekening;
  double _saldo;

  RekeningBank(this._nomorRekening, this._saldo);

  void deposit(double jumlah) {
    if (jumlah > 0) {
      _saldo += jumlah;
      print("Deposit berhasil: Rp$jumlah");
    }
  }

  void tarik(double jumlah) {
    if (jumlah > 0 && jumlah <= _saldo) {
      _saldo -= jumlah;
      print("Penarikan berhasil: Rp$jumlah");
    }
  }

  void cekSaldo() {
    print("Nomor Rekening: $_nomorRekening");
    print("Saldo saat ini: Rp$_saldo");
  }
}

void main() {
  RekeningBank rekening = RekeningBank("4567890", 400000);

  rekening.deposit(200000);
  rekening.tarik(100000);
  rekening.cekSaldo();
}
